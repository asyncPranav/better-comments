# Better Comments

Colorize your comments based on type — just like VS Code's Better Comments.

## ⭐ star the repo : [star here](https://github.com/asyncpranav/better-comments)

## Tags

| Tag | Color | Example |
|-----|-------|---------|
| `!` | 🔴 Red | `// ! This will crash` |
| `?` | 🔵 Blue | `// ? Is this correct?` |
| `*` | 🟢 Green | `// * Important section` |
| `TODO` | 🟠 Orange | `// TODO: fix this` |
| `FIXME` | 🔴 Red-Orange | `// FIXME: broken on Android` |
| `NOTE` | 🟣 Purple | `// NOTE: see the docs` |
| `//` | ⚫ Strikethrough | `// // dead code` |

## Supported Languages

Works in any file using `//` or `#` comments — JS, TS, Python, Shell, Java, C, CSS, and more.

## Settings

Open command palette → **Better Comments: Settings** to toggle each tag type on or off.
